# 🤖 AI功能完整使用指南

## 📝 功能概览

本系统已全面集成GLM大模型，为学生和教师提供智能化的学习和教学支持。以下是所有AI功能的详细说明。

---

## 🎓 学生端AI功能

### 1. 错题本智能分析

**位置**：学生 → 我的错题

**功能**：
- ✅ AI分析错题：自动分析错误原因、知识点缺陷和改进建议
- ✅ 向AI提问：针对错题进行交互式问答
- ✅ 推荐相似题：基于错题推荐相似练习题

**使用方法**：
1. 进入"我的错题"页面
2. 点击任意错题卡片上的"AI分析"按钮
3. 查看AI生成的详细分析报告
4. 可选择"向AI提问"进行深入学习
5. 点击"推荐相似题"获取练习题

**示例场景**：
```
学生做错了一道二次函数题目
→ 点击"AI分析"
→ AI分析出：1) 未掌握顶点式; 2) 图像平移概念模糊
→ 点击"向AI提问"："什么是顶点式？"
→ AI详细解答顶点式概念
→ 点击"推荐相似题"
→ 获得5道类似的练习题
```

---

### 2. 个人学习报告

**位置**：学生 → 个人中心

**功能**：
- ✅ AI生成个性化学习报告
- ✅ 分析学习进度和薄弱环节
- ✅ 提供改进建议和学习路径
- ✅ 下载报告到本地

**使用方法**：
1. 进入"个人中心"页面
2. 点击"AI学习报告"按钮
3. 等待AI分析（约3-5秒）
4. 查看详细的学习报告
5. 可点击"下载报告"保存到本地

**报告内容包括**：
- 📊 学习数据统计
- 📈 进步趋势分析
- ⚠️ 薄弱知识点识别
- 💡 个性化学习建议
- 🎯 下阶段学习计划

---

### 3. 知识点智能提取

**位置**：学生 → 知识点溯源

**功能**：
- ✅ 从错题中自动提取知识点
- ✅ 建立知识点关联
- ✅ 生成知识点树状结构

**使用方法**：
1. 进入"知识点溯源"页面
2. 选择学科（数学/语文/英语等）
3. 点击"AI提取知识点"按钮
4. AI自动从你的错题中提取知识点
5. 查看更新后的知识点树

**智能提取内容**：
- 🔍 知识点名称
- 📖 课本章节定位
- 🔗 关联题目
- 📝 难度等级

---

### 4. 共享池智能推荐

**位置**：学生 → 错题共享池

**功能**：
- ✅ 基于学习数据推荐优质内容
- ✅ 智能匹配薄弱知识点
- ✅ 个性化排序展示

**使用方法**：
1. 进入"错题共享池"页面
2. 点击"AI为你推荐"按钮
3. AI分析你的学习数据
4. 展示最适合你的共享错题
5. 查看AI推荐理由

**推荐依据**：
- 📚 你的错题分布
- 🎯 薄弱知识点
- 📈 学习进度
- 🔥 内容质量评分

---

### 5. 全局AI助手

**位置**：所有页面（右下角浮动按钮）

**功能**：
- ✅ 随时随地提问
- ✅ 学习问题解答
- ✅ 作业辅导
- ✅ 知识点讲解

**使用方法**：
1. 点击右下角的AI助手浮动按钮（蓝色机器人图标）
2. 在聊天框输入问题
3. 发送并等待AI回复
4. 支持多轮对话

**适用场景**：
```
- "一元二次方程的求根公式是什么？"
- "解释一下牛顿第三定律"
- "这道英语语法题怎么做？"
- "如何记忆化学元素周期表？"
```

## 💡 使用技巧

### 1. 提问技巧

**好的提问**：
- ✅ "二次函数的顶点式是什么？有什么应用？"
- ✅ "为什么这道题要用韦达定理？"
- ✅ "解释一下这个知识点的物理意义"

**不好的提问**：
- ❌ "帮我做作业"
- ❌ "答案是什么"
- ❌ 单字或过于简短的问题

### 2. AI分析最佳时机

- 📅 每周末生成学习报告
- 📝 做完作业后分析错题
- 🎯 考试前查看薄弱点
- 📊 月底进行全面诊断

### 3. 充分利用AI功能

**学习流程**：
```
做题 → 记录错题 → AI分析 → 向AI提问 → 做相似题 → 巩固提升
```

**复习流程**：
```
查看错题 → AI提取知识点 → 学习课本 → 生成学习报告 → 针对性练习
```

## ⚠️ 注意事项

1. **AI分析需要数据**：
   - 错题分析需要先添加错题
   - 学习报告需要至少5条错题数据
   - 班级分析需要学生提交数据

2. **网络要求**：
   - AI功能需要网络连接
   - 响应时间约2-5秒
   - 建议在网络良好时使用

3. **合理使用**：
   - AI是辅助工具，不能完全依赖
   - 建议结合教材和老师讲解
   - 培养独立思考能力

4. **隐私保护**：
   - 所有数据仅用于AI分析
   - 不会泄露个人信息
   - 符合数据安全规范

## 🆘 常见问题

### Q1: AI分析一直加载怎么办？
**A**: 检查网络连接，或刷新页面重试。如果持续失败，请联系技术支持。

### Q2: AI回答不准确怎么办？
**A**: 可以重新提问，提供更详细的问题描述。AI会根据上下文优化回答。

### Q3: 可以批量分析错题吗？
**A**: 可以！在个人中心生成学习报告时，AI会自动分析所有错题。

### Q4: AI生成的练习题可以保存吗？
**A**: 目前显示在弹窗中，建议截图保存或复制粘贴。

### Q5: 全局AI助手的对话会保存吗？
**A**: 当前对话会保存在当前会话中，关闭对话框后会清空。
