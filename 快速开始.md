## 🎯 5分钟快速启动

### 第一步: 安装Redis (可选但强烈推荐)

```bash

install-redis.bat

# 按提示选择安装方式:
# [1] Memurai (推荐) - 最简单,一键安装

# [3] 跳过Redis - 系统将降级运行(慢)
```

```bash
# 运行数据库优化脚本

```


## 🔍 常见问题

### Q1: Redis安装失败怎么办?

```
方案1: 使用Memurai (最简单)
1. 访问 https://www.memurai.com/get-memurai
2. 下载并安装
3. 重启 start-all.bat

方案2: 跳过Redis
1. 选择 [3] 跳过Redis
2. 系统将在无缓存模式下运行
3. 性能较差但能用
```

### Q2: 启动失败怎么办?

```bash
# 检查MySQL
sc query MySQL80
# 如果未运行,启动MySQL服务

# 检查Redis
check-redis.bat

# 查看错误日志
# 后端: backend/zszg-backend/logs/
# 前端: 控制台输出
```

### Q3: 性能很慢怎么办?

```bash
# 1. 确认Redis是否运行
check-redis.bat

# 2. 重新优化数据库
run-db-optimization.bat

# 3. 查看系统监控
curl http://localhost:8080/api/monitor/metrics
```
