==========================================
    "知错就改"演讲稿（精简版）
         时长：8-10分钟
==========================================


【开场】(30秒)
━━━━━━━━━━━━━━━━━━━━━━━━━━

各位老师、各位同学，大家好！

我是[您的姓名]，今天为大家介绍我们的项目
——"知错就改"智能错题管理系统。


【项目简介】(1分钟)
━━━━━━━━━━━━━━━━━━━━━━━━━━

"知错就改"是一个集成了AI图像识别、自然
语言处理和知识图谱的智能学习平台。

它有五大核心功能：
1. 拍照识别题目
2. AI智能分析
3. 知识图谱溯源
4. 协作学习共享
5. 数据可视化分析

系统服务于三类用户：学生、教师、管理员。


【技术架构】(2分钟)
━━━━━━━━━━━━━━━━━━━━━━━━━━

我们采用四层分层架构：

第一层，前端展示层
• Vue 3 + TypeScript + Element Plus
• Vite 5 + ECharts + Pinia

第二层，业务逻辑层
• Spring Boot 3.3.2
• Spring Security + JWT
• Spring Data JPA
• Java 17

第三层，AI服务层（核心竞争力）
• 智谱GLM-4V视觉模型（图像识别）
• GLM-4语言模型（智能分析）
• 知识图谱引擎
• 智能推荐算法

第四层，数据持久层
• MySQL 8.0

架构特点：
✅ 前后端分离，RESTful API
✅ JWT身份认证，RBAC权限
✅ AI微服务化
✅ 响应式设计


【学生端功能】(2分钟)
━━━━━━━━━━━━━━━━━━━━━━━━━━

一、智能错题本

📸 拍照识别
学生拍照上传题目，AI在3秒内自动识别，
准确率95%以上，支持数学公式识别。

🤖 AI智能分析
生成500字以上的详细解析：
• 第一步：理解题意
• 第二步：分析思路（解释"为什么"）
• 第三步：详细计算过程
• 错因诊断 + 改进建议

🌳 知识图谱溯源
构建三层知识体系：
• 前置知识点（基础）
• 当前知识点（核心）
• 后续知识点（拓展）
自动生成个性化学习路径！

二、协作学习

🌟 共享池
浏览优质解析，点赞、收藏、复制

💎 智能推荐
基于AI算法推荐：
• 相似错题
• 薄弱知识点题目
• 进阶题目


【教师端功能】(2分钟)
━━━━━━━━━━━━━━━━━━━━━━━━━━

一、数据分析面板

📊 全班学情统计
• 总错题数、平均订正率
• 学科分布（饼图）
• 难度分布（柱状图）
• 知识点热度（词云图）

🔍 AI智能分析
自动生成分析报告：
• 班级薄弱知识点TOP10
• 需要重点关注的学生
• 教学建议

二、智能任务发布

📝 自然语言描述
教师：
"本周五前完成第三章错题整理，至少5道题，
重点关注求导和极值问题"

AI自动解析：
✅ 任务类型：错题整理
✅ 截止时间：本周五
✅ 数量要求：至少5道
✅ 重点知识点：求导、极值
✅ 学习目标、难度评估

全都自动生成，老师只需确认！


【AI核心技术】(1分钟)
━━━━━━━━━━━━━━━━━━━━━━━━━━

   用户画像 → 知识点分析 → 题目匹配
