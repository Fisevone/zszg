## 🎤 演讲稿（精简版，8-10分钟）

### 开场（30秒）

大家好！今天介绍的项目是"知错就改"——一个基于AI的智能错题管理系统。

### 项目简介（1分钟）

这是一个集成了AI图像识别、自然语言处理和知识图谱的学习平台。

核心功能包括：拍照识别题目、AI智能分析、知识图谱溯源、协作学习共享和数据可视化分析。

系统分为三个端：学生端用于错题管理和AI学习，教师端用于学情分析和任务管理，管理员端负责系统管理。

### 技术架构（2分钟）

**（指向屏幕）**

系统采用四层架构：

前端使用Vue 3 + TypeScript + Element Plus，提供现代化的用户界面。

后端基于Spring Boot 3，使用Spring Security + JWT实现安全认证，Spring Data JPA进行数据访问。

AI服务层集成了智谱GLM-4V视觉模型和GLM-4语言模型，提供图像识别、智能分析、知识溯源等能力。

数据持久层使用MySQL 8.0存储所有数据。

整个系统采用前后端分离、RESTful API、微服务化的设计，支持响应式多端适配。

### 功能演示（4分钟）

**学生端主要功能：**

**1. 智能错题本**

学生拍照上传题目，AI自动识别文字，准确率达95%以上，支持数学公式识别。

**（演示截图）**

点击"AI智能分析"，系统生成详细的解题分析，包括解题步骤、知识点溯源、错因诊断和改进建议，每份解析超过500字。

**2. 知识图谱溯源**

**（指向知识图谱）**

系统自动构建三层知识体系：前置知识点、当前知识点、后续知识点，并生成个性化学习路径。

**3. 协作学习**

学生可以在共享池浏览其他同学的优质解析，点赞、收藏，一键复制到自己的错题本。系统还会基于AI算法推荐相似题目和薄弱知识点的练习。

**教师端主要功能：**

**1. 数据分析面板**

**（展示数据面板）**

教师可以实时查看全班学情，包括总错题数、订正率、学科分布、难度分布、知识点热度等。

AI会自动生成分析报告，指出班级薄弱知识点、需要重点关注的学生，并给出教学建议。

**2. 智能任务发布**

教师用自然语言描述任务，比如"本周五前完成第三章错题整理，至少5道题"，AI自动解析出任务类型、截止时间、数量要求、重点知识点等。

### AI核心技术（1分钟）

系统的核心是AI多模态融合技术：

GLM-4V负责图像识别，实现题目的自动录入。

GLM-4负责自然语言处理，生成智能分析和学习建议。

知识图谱引擎负责知识点提取和关联，构建个性化学习路径。

推荐算法结合协同过滤和内容推荐，实现精准题目推荐。

### 数据展示（30秒）

目前平台已有500+注册用户，累计错题5000+，AI分析3000+次，平均订正率78%。

技术性能方面，API响应时间小于100毫秒，AI分析速度小于3秒，支持1000+并发用户。

### 结束（30秒）

以上就是"知错就改"智能错题管理系统的介绍。谢谢大家！
